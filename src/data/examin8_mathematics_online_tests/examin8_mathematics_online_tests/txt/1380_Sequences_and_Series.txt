Q8. In an A.P., sum of first p terms is equal to the sum of first q terms. Sum of its first p + q terms is
    (A) pq
    (B) p + q
    (C) 0   <-- CORRECT
    (D) - (p + q)
    Answer: C
    Explanation: S_p = S_q
$\Rightarrow$ $\frac{p}{2}$(2a + (p - 1)d) = $\frac{q}{2}$(2a + (q - 1)d)
$\Rightarrow$ 2ap + (p^2 - p)d = 2aq + (q^2 - q)d
$\Rightarrow$ 2a(p - q) + (p^2 - p - q^2 + q) d = 0
$\Rightarrow$ 2a(p - q) + (p^2 - q - (p - q))d = 0
$\Rightarrow$ (p - q)(2a + (p + q - 1)d) = 0
$\Rightarrow$ (2a + ( p + q - 1)d) = 0
Now S_p+q = $\frac{p+q}{2}$ (2a(p + q - 1)d) = 0

Q10. In an A.P., sum of first p terms is q and sum of first q terms is p. Sum of its p + q terms is
    (A) p - q
    (B) - (p + q)   <-- CORRECT
    (C) 0
    (D) p + q
    Answer: B
    Explanation: We know that, S_n = $\frac{n}{2}$(2a + (n - 1)d)
S_p = $\frac{p}{2}$(2a + (p - 1)d) = q $\Rightarrow$ (2a + (p - 1)d) = $\frac {2q}p$ ...(i)
S_q = $\frac{q}{2}$(2a + (q - 1)d) = q $\Rightarrow$ (2a + (q - 1)d) = $\frac {2p}q$ ...(ii)
subtract equation(i) from (ii), we get
(q - p)d = $\frac {2p}{q} -\frac {2q}p$
$\Rightarrow$ (q - p)d = $\frac{2 p^{2}-2 q^{2}}{p q}=\frac{2(p-q)(p+q)}{p q}$
$\Rightarrow$ d = $\frac{-2(p+q)}{p q}$ ...(iii)
Now we have S_p + q = $\frac{p+q}2$[2a + (p + q - 1)d]
$\Rightarrow$ S_p + q = $\frac{p+q}2$[2a + (p - 1)d + qd]
$\Rightarrow$ S_p + q = $\frac{p+q}2\left[\frac{2 q}{p}+q\left(\frac{-2(p+q)}{p q}\right)\right]$ using (i) and (ii)
$\Rightarrow$ S_p + q = $\frac{p+q}2$(-2) = -(p + q)

    (A) x - 3

    (D) x

We have $S_{\infty}=\frac{a}{1-r}$ 
Given $S_{\infty}=\frac{5}{4}$. Sum of odd terms
$\Rightarrow$ a + ar + ar^2 + ar^3 + .... = $\frac{5}{4}$[a + ar^2 + ar^4 + ...]
$\Rightarrow$ $\frac{a}{1-r}=\frac{5}{4} \cdot \frac{a}{1-r^{2}}$ 
$\Rightarrow$ $\frac{1}{1-r}=\frac{5}{4} \cdot \frac{a}{(1-r)(1+r)}$ 
$\Rightarrow$ 4(1 + r) = 5
$\Rightarrow$ 4r = 1
$\Rightarrow$ r = $\frac{1}{4}$

Q10. If one A.M. A and two geometric means B and C be inserted between two numbers, then ${B^3} + {C^3}$ is equal to

Q11. In a G.P. the ratio of the sum of first three terms to the sum of first six terms is 125 : 152. The common ratio of the G.P. is
    (A) 3.5
    (B) $\frac {3}{5}$   <-- CORRECT
    (C) 5.3
    (D) $\frac 53$
    Answer: B
    Explanation: Given $\frac{S_{3}}{S_{6}}=\frac{125}{152}$ 
$\Rightarrow$ $\frac{\frac{a\left(r^{3}-1\right)}{r-1}}{\frac{a\left(r^{6}-1\right)}{r-1}}=\frac{125}{152}, r-1 \neq 0$ 
$\Rightarrow$ $\frac{r^{3}-1}{r^{6}-1}=\frac{125}{152}$ 
$\Rightarrow$ 152r^3 - 15^2 = 125r^6 - 125
$\Rightarrow$ 125r^6 - 152r^3 + 27 = 0
$\Rightarrow$ 125r^6 - 125r^3 - 27r^3 + 27 = 0
$\Rightarrow$ 125r^3(r^3 - 1) - 27(r^3 - 1) = 0
$\Rightarrow$ (125r^3 - 27)(r^3 - 1) = 0

Since r - 1 $\ne$ 0, r cannot be 1

Q12. In a G.P. (m + n)^th and (m – n)^th terms are respectively p and q. Its m^th term is
    (A) pq

Now dividing equation (i) by (ii)
$\frac { ar^{ m+n-1 } }{ ar^{ m-n-1 } } =\frac { p }{ q } \Rightarrow$ r^(m + n - 1) - (m - n - 1) = $\frac pq$
$\Rightarrow$ $r2n = \frac { p }{ q }$

Substituting equation (iii) in (i), we get 
$a.\sqrt { \frac { p }{ q } } .r^{ m-1 }$ = p
$\Rightarrow a.r^{ m-1 }=p.\sqrt { \frac { q }{ p } } =\sqrt { pq } $
