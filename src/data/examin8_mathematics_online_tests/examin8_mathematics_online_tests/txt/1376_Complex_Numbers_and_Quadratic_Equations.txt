Q1. The complex numbers z = x + iy which satisfy the equation $\left|\frac{z-3 i}{z+3 i}\right|=1$ lie on
    (A) on x axis.
    (B) the straight line y = 3
    (C) the x-axis   <-- CORRECT
    (D) a circle passing through the origin
    Answer: C
    Explanation: Given$\left|\frac{z-3 i}{z+3 i}\right|=1$
Putting z = x + iy , we get
$\left|\frac{x+i y-3 i}{x+i y+3 i}\right|=1$
$\Rightarrow\left| {{{x+iy - 3i} }} \right| =\left| {{{x+iy + 3i} }} \right|$
$\Rightarrow\left| {{{x+i(y - 3)} }} \right| =\left| {{{x+i(y + 3)} }} \right|$
$\Rightarrow \sqrt { { x }^{ 2 }+{ \left( { y-3 } \right) }^{ 2 } } =\sqrt { { x }^{ 2 }+{ \left( { y+3 } \right) }^{ 2 } } $ $\left[ \because \left| x+iy \right| =\sqrt { { x }^{ 2 }+{ y }^{ 2 } } \right]$
Squaring on both sides, we get
${ x }^{ 2 }+{ \left( { y-3 } \right) }^{ 2 }={ x }^{ 2 }+{ \left( { y+3 } \right) }^{ 2 }$
$ \Rightarrow { x }^{ 2 }+{ y }^{ 2 }-6y+9={ x }^{ 2 }+{ y }^{ 2 }+6y+9$
$ \Rightarrow 12y=0$
$ \Rightarrow y=0$
Which means the X-axis.

    (A) Re(z) $<$ 0
    (B) y > 4
    (C) Re(z) > 0
    (D) x > 3   <-- CORRECT

$\Rightarrow$ |(x - 4) + iy| < |(x - 2) + iy| 
$\Rightarrow$ $\sqrt{(x-4)^{2}+y^{2}}$ < $\sqrt{(x-2)^{2}+y^{2}}$ [$\because$ |x + iy| = $\sqrt{x^{2}+y^{2}}$]

(x - 4)^2 + y^2 < (x - 2)^2 + y^2

$\Rightarrow$ 16 - 4 < 8x - 4x
$\Rightarrow$ 12 < 4x
$\Rightarrow$ x > 3

$\Rightarrow$ Z^3 = $\frac{8 i}{8}$ = i
