$\Rightarrow { { { \frac { 2x-1 }{ x-1 } >2 } } }\quad or\quad \frac { 2x-1 }{ x-1 } <-2\quad \quad \quad \quad \quad \left[ \because \left| x \right| >a\Rightarrow x<-a\quad or\quad x>a \right] $

$ \frac { 2x-1 }{ x-1 } -2>0$
$\Rightarrow \frac{2 x-1-2(x-1)}{x-1}>0$
$\Rightarrow \frac{2 x-1-2 x+2}{x-1}>0$
$\Rightarrow \frac{1}{x-1}>0$
$\Rightarrow x-1>0$
$\Rightarrow x>1$
$\Rightarrow x \epsilon(1, \infty)$
Now $ \frac { 2x-1 }{ x-1 } <-2$
$\Rightarrow \frac{2 x-1}{x-1}+2<0$
$\Rightarrow \frac{2 x-1+2(x-1)}{x-1}<0$

$\Rightarrow(4 x-3>0 \text { and } x-1<0) \text { or }(4 x-3<0 \text { and } x-1>0)$
$\left[\because \frac{a}{b}<0 \Rightarrow(a>0 \text { and } b<0) \quad \text { or } \quad(a<0 \text { and } \quad b>0)\right]$
$\Rightarrow\left(x>\frac{3}{4} \quad \text { and } \quad x<1 \quad\right) \quad \text { or } \quad\left(x<\frac{3}{4} \quad \text { and } \quad x>1\right)$
$\left[\text { Since } \quad x<\frac{3}{4} \quad \text { and } \quad x>1 \text { is } \quad \text { not } \quad \text { possible }\right]$
$\Rightarrow x \in\left(\frac{3}{4}, 1\right)$

    (A) ( -2 , 3)
    (B) ( - 4 , 3 )
    (C) ( 1 , 3 )
    (D) no solution   <-- CORRECT
    Answer: D
    Explanation: $- {\text{ 4x }} + {\text{ 1 }} \geq {\text{ }}0{\text{ }}$
$\Rightarrow -4x\ge -1\\ \Rightarrow \frac { -4x }{ -4 } \le \frac { -1 }{ -4 } \\ \Rightarrow x\le \frac { 1 }{ 4 } \\ \Rightarrow x\epsilon \left( -\infty ,\frac { 1 }{ 4 } \right]$
${\text{ 3 }}-{\text{ 4x }} < {\text{ }}0$
$\Rightarrow -4x<-3\\ \Rightarrow \frac { -4x }{ -4 } >\frac { -3 }{ -4 } \\ \Rightarrow x>\frac { 3 }{ 4 } \\ \Rightarrow x\epsilon \left( \frac { 3 }{ 4 } ,\infty \right) $
Hence solution set is $\left( -\infty ,\frac { -5 }{ 2 } \right] \bigcap { \left( \frac { 3 }{ 4 } ,\infty \right) } =\Phi $
which means no solution exist.

Q2. Solve the system of inequalities: x - 5 > 0, $\frac{2 x-4}{x+2}$ < 2
    (A) x > -5
    (B) x > 2
    (C) x < -2
    (D) x > 5   <-- CORRECT
    Answer: D
    Explanation: x - 5 > 0
$\Rightarrow$ x > 5
$\Rightarrow$ $x \in(5, \infty)$ 
Now $\frac{2 x-4}{x+2}$ < 2
$\frac{2 x-4}{x+2}$ - 2 < 0
$\Rightarrow$ $\frac{2 x-4-2(x+2)}{x+2}$ < 0
$\Rightarrow$ $\frac{2 x-4-2 x-4}{x+2}$ < 0
$\Rightarrow$ $\frac{-8}{x+2}$ < 0
$\Rightarrow$ x + 2 > 0 [$\because$ $\frac{a}{b}$ < 0, a < 0 $\Rightarrow$ b > 0]
$\Rightarrow$ x > -2
$\Rightarrow$ $x \in(-2, \infty)$ 
Hence the solution set is $\left( 5,\infty \right) \cap \left( -2,\infty \right) =\left( 5,\infty \right) $ 
Which means x > 5.

Q3. Solve the system of inequalities: $\frac{x+7}{x-8}>2, \frac{2 x+1}{7 x-1}>5$
    (A) no solution   <-- CORRECT
    (B) (2, 5)
    (C) (3, 6)
    (D) (4, 8)
    Answer: A
    Explanation: $\frac {x + 7} {x - 8} > 2$ 
$\Rightarrow \frac {x + 7} {x - 8} -2> 0$
$\Rightarrow$ $\frac{x+7-2(x-8)}{x-8}$ > 0
$\Rightarrow$ $\frac{x+7-2 x+16}{x-8}$ > 0
$\Rightarrow$ $\frac{(23-x)}{x-8}$ > 0 [$\because$ $\frac{a}{b}$ > 0 $\Rightarrow$ (a > 0 and b > 0) or (a < 0 and b < 0)]
$\Rightarrow$ (23 - x > 0 and x - 8 > 0) or (23 - x < 0 and x - 8 < 0)
$\Rightarrow$ (x < 23 and x > 8) or (x > 23 and x < 8)
$\Rightarrow$ 8 < x < 23 [Since x > 23 and x < 8 is not possible]
$\Rightarrow x\epsilon \left( 8,23 \right) $
Now ${{2x + 1} \over {7x - 1}}$ > 5 
$\Rightarrow \frac {2x + 1} {7x - 1}$ - 5 > 0
$\Rightarrow$ $\frac{2 x+1-5(7 x-1)}{7 x-1}$ > 0
$\Rightarrow$ $\frac{2 x+1-35 x+5}{7 x-1}$ > 0
$\Rightarrow$ $\frac{(6-33 x)}{7 x-1}$ > 0 [$\because$ $\frac{a}{b}$ > 0 $\Rightarrow$ (a > 0 and b > 0) or (a < 0 and b < 0)]
$\Rightarrow$ (6 - 33x > 0 and 7x - 1 > 0) or (6 - 33x < 0 and 7x - 1 < 0)
$\Rightarrow$ (x < $\frac{6}{33}$ and x > $\frac{1}{7}$) or (x > $\frac{2}{11}$ and x < $\frac{1}{7}$)
$\Rightarrow$ $\frac{1}{7}<x<\frac{2}{11}$ 
$\Rightarrow$ $x \in\left(\frac{1}{7}, \frac{2}{11}\right)$ [Since x > $\frac{2}{11}$ and x < $\frac{1}{7}$ is not possible]
Hence, the solution of the system $\frac{x+7}{x-8}>2, \frac{2 x+1}{7 x-1}>5$ will be $\left( 8,23 \right) \cap \left( \frac { 1 }{ 7 } ,\frac { 2 }{ 11 } \right) =\phi $

Q11. Find all pairs of consecutive even positive integers, both of which are larger than 5, such that their sum is less than 23.
    (A) (8, 6), (6, 4), (10, 6)
    (B) (6, 8),(8, 10), (10, 12)   <-- CORRECT
    (C) (4, 6), (6, 8), (8, 10)
    (D) (3, 5), (5, 7), (7, 9)
    Answer: B
    Explanation: Let the consecutive even positive integers be x and x + 2.
By data, x > 5 and x + (x + 2) < 23
Now x + (x + 2) < 23
$\Rightarrow$ 2x + 2 < 23
$\Rightarrow$ 2x < 21
$\Rightarrow$ x < $\frac{21}{2}$ = $10\frac{1}{2}$
So we have the least possible value of x is 6 and the maximum value of x is 10.
Therefore the possible pairs of consecutive even positive integers are (6, 8), (8, 10), (10, 12).

    (A) {0, 1, 2, 3}   <-- CORRECT

    (D) {1, 2, 3}
    Answer: A
    Explanation: Given 2(x - 1) < 3x - 1 
$\Rightarrow$ 2x - 2 < 3x - 1
$\Rightarrow$ 2x - 2 + 2 < 3x - 1 + 2
$\Rightarrow$ 2x < 3x + 1
$\Rightarrow$ 2x - 3x < 3x + 1 - 3x 
$\Rightarrow$ -x < + 1
$\Rightarrow$ x > -1 but x $\in$ Z
Hence A = {0, 1, 2, 3, 4, ....}
Now 4x - 3 $\leq$ 8 + x
$\Rightarrow$ 4x - 3 + 3 $\leq$ 8 + x + 3
$\Rightarrow$ 4x $\leq$ 11 + x
$\Rightarrow$ 4x - x $\leq$ 11 + x - x
$\Rightarrow$ 3x $\leq$ 11
$\Rightarrow$ $\frac{3x}{3}$ $\leq$ $\frac{11}{3}$ 
$\Rightarrow$ x $\leq$ $\frac{11}{3}$ 
$\Rightarrow$ x $\leq$ $3\frac{2}{3}$, but x $\in$ Z
