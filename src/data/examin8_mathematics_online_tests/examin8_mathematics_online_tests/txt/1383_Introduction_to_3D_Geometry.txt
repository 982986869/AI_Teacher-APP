Q9. The radius of the sphere through the points (4, 3, 0), (0, 4, 3), (0, 5, 0) and (4, 0, 3) is
    (A) 5   <-- CORRECT
    (B) $\frac{5}{7}$
    (C) 7
    (D) $\frac{7}{5}$
    Answer: A
    Explanation: We know that general equation of sphere is x^2 + y^2 + z^2 + 2gx + 2fy + 2hz + c = 0
Since sphere passing through the points (4, 3, 0), (0, 4, 3), (0, 5, 0) and (4, 0, 3) so putting these values one by one in given equation, we get
16 + 9 + 8g + 6f + c = 0 $\Rightarrow$ 8g + 6f + c = -25 ......(i)
16 + 9 + 8f + 6h + c = 0 $\Rightarrow$ 8f + 6h + c = -25 ......(ii)
25 + 10f + c = 0 ......(iii)
16 + 9 + 8g + 6h + c = 0 $\Rightarrow$ 8g + 6h + c = -25 ......(iv)
By (ii) - (iv) we have f = g
Using this in (i) we have 16f +c = -25 .....(v)
Now solving (iii) and (v) we get f = 0 
Using value of f we have g = 0, h = 0, c = -25
Now radius =$\sqrt {{{( - g)}^2} + ( - f) + {{( - h)}^2} - c} $ = $\sqrt {25} $ = 5

Q14. An angle between two diagonals of a cube is
    (A) cos^-1 $\left(\frac{1}{\sqrt{2}}\right)$
    (B) cos^-1 $\left(\frac{8}{5}\right)$
    (C) cos^-1 $\left(\frac{9}{5}\right)$
    (D) cos^-1 $\left(\frac{1}{3}\right)$   <-- CORRECT
    Answer: D
    Explanation: Let a be the length of the cube and one corner D(0,0,0) as the origin.the diagonals will have co-ordinates as follows:
A(a,0,0) G(0,a,a) ie diagonal AG and diagonal DF where F(a,a,a)
$cos\theta=\frac{a X -a+a Xa+a X a}{ \sqrt {-a^2+a^2+a^2}\sqrt {a^2+a^2+a^2}}$
$cos \theta =\frac{-a^2+a^2+a^2}{\sqrt {3a^2}\sqrt {3a^2}}$
cos^-1 = $\left(\frac{1}{3}\right)$
