Q1. In a triangle ABC, tan $\frac{A}{2}=\frac{5}{6}, \tan \frac{B}{2}=\frac{20}{37}$ then tan $\frac C 2$ is equal to
    (A) $\frac {100} {222}$
    (B) $\frac 2 5$   <-- CORRECT
    (C) $\frac {34} {37}$
    (D) $\frac 5 2$
    Answer: B
    Explanation: Given tan $\frac{A}{2}$ = $\frac{5}{6}$ $\Rightarrow$ cot $\frac{A}{2}$ = $\frac{6}{5}$, tan $\frac{B}{2}$ = $\frac{20}{37}$ $\Rightarrow$ cot $\frac{B}{2}$ = $\frac{37}{20}$ 
Given A, B and C are angles of a triangle 
Then, A + B + C = 180
$\Rightarrow$ $\frac{A+B+C}{2}$ = $\frac{180}{2}$ 
$\Rightarrow$ $\frac{A+B}{2}+\frac{C}{2}$ = 90
$\Rightarrow$ $\frac{C}{2}$ = $90-\frac{A+B}{2}$ 
Taking tan on both sides, we get
$\Rightarrow$ $\tan \frac{C}{2}$ = $\tan \left(90-\frac{A+B}{2}\right)$ 
$\Rightarrow$ $\tan \frac{C}{2}$ = $\cot \left(\frac{A+B}{2}\right)$ = $\cot \left(\frac{A}{2}+\frac{B}{2}\right)$ = $\frac{\cot \frac{A}{2} \cdot \cot \frac{B}{2}-1}{\cot \frac{A}{2}+\cot \frac{B}{2}}$ = $\frac{\frac{6}{5} \cdot \frac{37}{20}-1}{\frac{6}{5}+\frac{37}{20}}$ = $\frac{\frac{222}{100}-1}{\frac{305}{100}}$ = $\frac{\frac{122}{100}}{\frac{305}{100}}$ = $\frac{122}{305}$ = $\frac{2}{5}$

Q9. In a triangle ABC, $\tan \frac{A+B}{2} \cot \frac{A-B}{2}$ is equal to
    (A) $\frac {a + b} {a - b}$   <-- CORRECT
    (B) $\frac {a - b} {c}$
    (C) $\frac {a - b} {a + b}$
    (D) $\frac {a + b} {c}$
    Answer: A
    Explanation: We have ${ tan\left( \frac { A-B }{ 2 } \right) }=\frac { a-b }{ a+b } .cot\frac { C }{ 2 } $
But in a triangle ABC, we have A + B + C = ${ 180 }^{ \circ }\Rightarrow \frac { C }{ 2 } ={ 90 }^{ \circ }-\left( \frac { A+B }{ 2 } \right) $
$\therefore \tan \left(\frac{A-B}{2}\right)=\frac{a-b}{a+b} \cdot \cot \left[90^{\circ}-\left(\frac{A+B}{2}\right)\right]$
$\Rightarrow \tan \left(\frac{\dot{A}-B}{2}\right)=\frac{a-b}{a+b} \cdot \tan \left(\frac{A+B}{2}\right)$
$\Rightarrow \tan \left(\frac{A+B}{2}\right) \cdot \cot \left(\frac{A-B}{2}\right)=\frac{a+b}{a-b}$

    (D) ${(s - b)^2}$

$=b c\left(\frac{1+\cos A}{2}\right)+c a\left(\frac{1+\cos B}{2}\right)+a b\left(\frac{1+\cos C}{2}\right)$
$=b c\left(\frac{1+\left(\frac{b^{2}+c^{2}-a^{2}}{2 b c}\right)}{2}\right)+c a\left(\frac{1+\left(\frac{c^{2}+a^{2}-b^{2}}{2 c a}\right)}{2}\right)+a b\left(\frac{1+\left(\frac{a^{2}+b^{2}-c^{2}}{2 a b}\right)}{2}\right)$
$=b c\left(\frac{2 b c+b^{2}+c^{2}-a^{2}}{4 b c}\right)+c a\left(\frac{2 c a+c^{2}+a^{2}-b^{2}}{4 c a}\right)+a b\left(\frac{2 a b+a^{2}+b^{2}-c^{2}}{4 a b}\right)$
$=\frac{2 b c+b^{2}+c^{2}-a^{2}+2 c a+c^{2}+a^{2}-b^{2}+2 a b+a^{2}+b^{2}-c^{2}}{4}$
$=\frac{a^{2}+b^{2}+c^{2}+2 b c+2 c a+2 a b}{4}$
$=\left(\frac{a+b+c}{2}\right)^{2} \quad[\because \quad 2 s=a+b+c]$
$=s^{2}$

Q15. If the sides of a triangle be 2x + 1, x^2 - 1 and x^2 + x + 1, then the greatest angle is
    (A) 90^o
    (B) 135^o
    (C) 150^o
    (D) 120^o   <-- CORRECT
    Answer: D
    Explanation: Let the sides of the triangle be a = 2x + 1, b = x^2 - 1 and c = x^2 + x + 1
We have cos C = $\frac{a^{2}+b^{2}-c^{2}}{2 a b}$ = $\frac{(2 x+1)^{2}+\left(x^{2}-1\right)^{2}-\left(x^{2}+x+1\right)^{2}}{2(2 x+1)\left(x^{2}-1\right)}$ 
= $\frac{4 x^{2}+4 x+1+x^{4}-2 x^{2}+1-\left(x^{4}+x^{2}+1+2 x^{3}+2 x^{2}+2 x\right)}{2\left(2 x^{3}-2 x+x^{2}-1\right)}$ 
= $\frac{-2 x^{3}+2 x-x^{2}+1}{2\left(2 x^{3}-2 x+x^{2}-1\right)}$ = $\frac{-1}{2}$ 
Now cos C = $\frac{-1}{2}$ 
But we have $\cos \left(\frac{2 \pi}{3}\right)$ = $\cos \left(\pi-\frac{\pi}{3}\right)$ = $-\cos \left(\frac{\pi}{3}\right)$ = $\frac{-1}{2}$ 
$\therefore$ C = $\frac{2 \pi}{3}$ = 120^o

Q1. If in a triangle ABC, $a{\cos ^2}\left( {{\frac C2}} \right) + c{\cos ^2}\left( {{\frac A2}} \right) = {\frac {3b} {2}}$ then its sides a, b, c are in
    (A) A. P.   <-- CORRECT
    (B) G.P.
    (C) H.P.
    (D) H.M
    Answer: A
    Explanation: $a \cos ^{2}\left(\frac{C}{2}\right)+c \cos ^{2}\left(\frac{A}{2}\right)$ = $\frac{3 b}{2}$ 
$\Rightarrow$ $a\left(\frac{1+\cos C}{2}\right)+c\left(\frac{1+\cos A}{2}\right)$ = $\frac{3b}{2}$ 
$\Rightarrow$ $\frac{1}{2}$ (b + a + c) = $\frac{3b}{2}$ [$\because$ a cos C + c cos A = b using projection rule]
$\Rightarrow$ a + c = 2b
Which means they are in A.P.

    (A) right angled
    (B) isosceles   <-- CORRECT
    (C) equilateral
    (D) right angled isosceles
    Answer: B
    Explanation: Given a sin A = b sin B .....(i)
But using sine rule we have $\frac{a}{\sin A}=\frac{b}{\sin B}=\frac{c}{\sin C}$ = k
$\therefore$ sin A = $\frac{a}{k}$ and sin B = $\frac{b}{k}$ 
Now from (i) we get $\frac{1}{k} $(a^2 - b^2) = 0 $\Rightarrow$ (a + b) (a - b) = 0 $\Rightarrow$ a = b

Q3. If in a $\Delta ABC$ , a cos A = b cos B, then the triangle is
    (A) either isosceles or right angled   <-- CORRECT
    (B) right angled
    (C) equilateral
    (D) isosceles
    Answer: A
    Explanation: a.cos A=b.cos B
$\Rightarrow a \cdot\left(\frac{b^{2}+c^{2}-a^{2}}{2 b c}\right)=b \cdot\left(\frac{c^{2}+a^{2}-b^{2}}{2 c a}\right)$
$\Rightarrow \frac{a^{2}\left(b^{2}+c^{2}-a^{2}\right)-b^{2}\left(c^{2}+a^{2}-b^{2}\right)}{2 a b c}=0$
$\Rightarrow a^{2} c^{2}-b^{2} c^{2}=a^{4}-b^{4}$
$\Rightarrow c^{2}\left(a^{2}-b^{2}\right)=\left(a^{2}-b^{2}\right)\left(a^{2}+b^{2}\right)$
$\Rightarrow a^{2}-b^{2}=0 \quad \text { or } \quad a^{2}+b^{2}=c^{2}$$\Rightarrow a=b \quad \text { or } \quad a^{2}+b^{2}=c^{2}$
Hence the triangle is either isosceles or right angle

Q4. In a $\Delta ABC$, $\frac{\cos ^2\left(\frac{A}{2}\right)}{a}+\frac{\cos ^2\left(\frac{B}{2}\right)}{b}+\frac{\cos ^2\left(\frac{C}{2}\right)}{c}$ is equal to
    (A) $\frac {2s} {abc}$
    (B) $\frac {3s} {abc}$
    (C) $\frac {s} {abc}$
    (D) ${\frac {s^2} {abc}}$   <-- CORRECT
    Answer: D
    Explanation: $\frac{\cos ^{2}\left(\frac{A}{2}\right)}{a}+\frac{\cos ^{2}\left(\frac{B}{2}\right)}{b}+\frac{\cos ^{2}\left(\frac{c}{2}\right)}{c}$ = $\frac{\frac{1+\cos A}{2}}{a}+\frac{\frac{1+\cos B}{2}}{b}+\frac{\frac{1+\cos C}{2}}{c}$ 
= $\frac { \frac { 1+\left( \frac { { b }^{ 2 }+{ c }^{ 2 }-a^{ 2 } }{ 2bc } \right) }{ 2 } }{ a } +\frac { \frac { 1+\left( \frac { { c }^{ 2 }+a^{ 2 }-{ b }^{ 2 } }{ 2ca } \right) }{ 2 } }{ b } +\frac { \frac { 1+\left( \frac { { a^{ 2 }+b }^{ 2 }-{ c }^{ 2 } }{ 2ab } \right) }{ 2 } }{ c }$ 
= $\frac{2 b c+b^{2}+c^{2}-a^{2}+2 c a+c^{2}+a^{2}-b^{2}+2 a b+a^{2}+b^{2}-c^{2}}{4 a b c}$ 
= $\frac{a^{2}+b^{2}+c^{2}+2 b c+2 c a+2 a b}{4 a b c}$ 
= $\frac{\left(\frac{a+b+c}{2}\right)^{2}}{a b c}$ [$\because$ 2s = a + b + c]
= $\frac{s^{2}}{a b c}$

Q5. In a triangle ABC, a = 2b and $\angle A = 3\angle B,$ then angle A is
    (A) 30^o
    (B) 60^o
    (C) 45^o
    (D) 90^o   <-- CORRECT
    Answer: D
    Explanation: Using Sine rule we have $\frac{a}{\sin A}=\frac{b}{\sin B}=\frac{c}{\sin C}$ = k
But given a = 2b and $\angle A=3 \angle B$ 
$\therefore$ $\frac{a}{\sin A}=\frac{b}{\sin B}$ $\Rightarrow$ $\frac{2 b}{\sin 3 B}=\frac{b}{\sin B}$ 
$\Rightarrow$ sin 3B = 2 sin B
$\Rightarrow$ 3 sin B - 4 sin^3 B = 2 sin B
$\Rightarrow$ sin B (1 - 4 sin^2 B) = 0
$\Rightarrow$ sin^2 B = $\frac{1}{4}$ [$\because$ sin B $\ne$ 0]
$\Rightarrow$ sin B = $\frac{\pm 1}{2}$ 
Now sin B = $\frac{1}{2}$ $\Rightarrow$ B = 30^o and sin B = $\frac{-1}{2}$ $\Rightarrow$ B = 210^o, which is not possible.
$\Rightarrow$ B = 30^o
Hence $\angle A=3 \angle B$ = 90^o
